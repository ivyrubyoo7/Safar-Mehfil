# Safar Mehfil

*A Travel Explorer Platform with Interactive Guidebook and Memory Tracking*

---

## 📌 Overview

**Safar Mehfil** is a modern, responsive web application designed to provide users with an immersive travel exploration experience. The platform combines a traditional travel discovery interface with a unique **interactive guidebook (flipbook-style UI)** and a **personal travel memory system**, enabling users to both explore destinations and document their journeys.

The application is built using **HTML, CSS, and vanilla JavaScript**, focusing on performance, responsiveness, and clean user experience without relying on external frameworks.

---

* Demo Link🔗: https://safar-mehfil.netlify.app/
## 🎯 Objectives

* Provide a visually engaging platform for discovering global travel destinations
* Deliver a unique storytelling experience through an interactive guidebook
* Enable users to save destinations and track visited places
* Maintain a clean, responsive, and mobile-first design architecture

---

## 🚀 Key Features

### 🧭 Navigation System

* Responsive and adaptive navbar
* Mobile-friendly hamburger menu
* Seamless navigation across all pages

---

### 🏠 Landing Page

* Hero section with call-to-action
* Featured destinations
* Category-based browsing (Beaches, Mountains, Cities, Culture)
* Trending destinations section

---

### 🔍 Explore Page

* E-commerce style destination listing
* Responsive card-based layout
* Destination details:

  * Image
  * Name
  * Location (City, Country)
  * Rating
  * Category tags
* UI-based filtering and search

---

### 📖 Guidebook (Flipbook Feature)

* Interactive book-style interface
* Smooth page flip animations using CSS and JavaScript
* Each page contains:

  * Destination details (city, state, country, region)
  * Description
  * Categories (Tourism, Culture, Food, etc.)
  * Best places to visit
* Keyboard and button navigation
* Mobile fallback with vertical layout

---

### 📄 Destination Details Page

* Detailed destination overview
* Image/banner section
* Location hierarchy
* Highlighted attractions
* Action buttons (Wishlist, Mark as Visited)

---

### ❤️ Wishlist System

* Add/remove destinations
* Persistent storage using localStorage
* Grid-based display of saved items

---

### 📸 Memories (Milestone System)

* Users can mark destinations as “Visited”
* Upload personal images
* Add notes and visit dates
* Scrapbook-style card layout
* Stored using localStorage
* Represents personal travel milestones

---

### 🔐 Authentication (UI Simulation)

* Login and registration forms
* Toggle between login/signup
* Clean and minimal UI design
